# CMDJ
An easy judge system on CMD.
Hope you feel covenient to this project.

1. Write your code in file 'main.cpp'
2. Write your input in file 'io/00.in'
3. Write the currect output in file 'io/00.ans'
4. Add more test case to file 'io/01.in','io/01.ans','io/02.in'... if you want
5. Run python c.py'

ENJOY!
